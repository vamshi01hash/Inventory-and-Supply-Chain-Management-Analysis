# Inventory & Supply Chain Management Analysis

This project presents a Power BI dashboard designed to analyze key areas of inventory and supply chain operations. It provides insights into stock levels, supplier performance, demand trends, and SKU movement to help businesses make informed decisions on procurement, warehousing, and replenishment.

## Key Features
- Visualization of total stock, aging inventory, and stockout risks  
- Supplier lead time analysis and on-time delivery performance  
- Fast-moving vs slow-moving SKU classification  
- Monthly demand trends and category-level insights  
- Interactive filters for supplier, product, region, and warehouse  

## Tools & Technologies
- Power BI (Data Modeling, DAX, Visualization)  
- Power Query for data transformation  
- Excel/CSV data sources  

## Project Benefits
- Improves inventory planning and reduces overstock or shortage issues  
- Identifies unreliable suppliers and potential delivery delays  
- Enhances understanding of demand cycles for better forecasting  

## How to Use
Download the `.pbix` file and open it in Power BI Desktop to explore and interact with the dashboard.
